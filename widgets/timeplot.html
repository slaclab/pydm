class pydm.widgets.timeplot.PyDMTimePlot(parent=None, init_y_channels=[], plot_by_timestamps=True, background='default', bottom_axis=None)[source]ÔÉÅ

Bases: BasePlot, updateMode

PyDMTimePlot is a widget to plot one or more channels vs. time.

Each curve can plot either a Y-axis waveform vs. its indices, or a Y-axis waveform against an X-axis waveform.

Parameters:

  • parent (optional) ‚Äì The parent of this widget.

  • init_y_channels (list) ‚Äì A list of scalar channels to plot vs time.

  • plot_by_timestamps (bool) ‚Äì If True, the x-axis shows timestamps as ticks, and those timestamps scroll to the left as time progresses. If False, the x-axis tick marks show time relative to the current time.

  • background (optional) ‚Äì The background color for the plot. Accepts any arguments that pyqtgraph.mkColor will accept.

  • bottom_axis (AxisItem, optional) ‚Äì Will set the bottom axis of this plot to the input axis. If not set, will default to either a TimeAxisItem if plot_by_timestamps is true, or a regular AxisItem otherwise

class pydm.widgets.timeplot.TimePlotCurveItem(channel_address=None, plot_by_timestamps=True, plot_style='Line', **kws)[source]ÔÉÅ

Bases: BasePlotCurveItem

TimePlotCurveItem represents a single curve in a time plot.

It is used to plot a scalar value vs. time. In addition to the parameters listed below, TimePlotCurveItem accepts keyword arguments for all plot options that pyqtgraph.PlotDataItem accepts.

Parameters:

  • channel_address (str) ‚Äì The address to of the scalar data to plot.

  • plot_by_timestamps (bool) ‚Äì If True, the x-axis shows timestamps as ticks, and those timestamps scroll to the left as time progresses. If False, the x-axis tick marks show time relative to the current time.

  • plot_style (str, optional) ‚Äì Currently one of either ‚ÄòLine‚Äô or ‚ÄòBar‚Äô. Determines how data points for this curve will be plotted. Defaults to a line based plot if not set

  • color (QColor, optional) ‚Äì The color used to draw the curve line and the symbols.

  • lineStyle (int, optional) ‚Äì Style of the line connecting the data points. Must be a value from the Qt::PenStyle enum (see http://doc.qt.io/qt-5/qt.html#PenStyle-enum).

  • lineWidth (int, optional) ‚Äì Width of the line connecting the data points.

  • **kws (dict) ‚Äì Additional parameters supported by pyqtgraph.PlotDataItem, like ‚Äòsymbol‚Äô and ‚ÄòsymbolSize‚Äô.

receiveNewValue(new_value)[source]ÔÉÅ

Rotate and fill the data buffer when a new value is available.

For Synchronous mode, write the new value into the data buffer immediately, and increment the accumulated point counter. For Asynchronous, write the new value into a temporary (buffered) variable, which will be written to the data buffer when asyncUpdate is called.

This method is usually called by a PyDMChannel when it updates. You can call it yourself to inject data into the curve.

Parameters:

new_value (float) ‚Äì The new y-value.
