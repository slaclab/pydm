class pydm.widgets.logdisplay.PyDMLogDisplay(parent=None, logname=None, level=0)[source]

Bases: QWidget

Standard display for Log Output

This widget handles instantiating a GuiHandler and displaying log messages to a QPlainTextEdit. The level of the log can be changed from inside the widget itself, allowing users to select from any of the .levels specified by the widget.

Parameters:

  • parent (QObject, optional) –

  • logname (str) – Name of log to display in widget

  • level (logging.Level) – Initial level of log display
