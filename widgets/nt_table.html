The PyDMNTTable is a table widget used to display PVA NTTable data.

The PyDMNTTable has two ways of filling the table from the data. If the incoming data dictionary has a ‚Äòlabels‚Äô and/or a ‚Äòvalue‚Äô key. Then the list of labels will be set with the data from the ‚Äòlabels‚Äô key. While the data from the ‚Äòvalue‚Äô key will be used to set the values in the table. if neither ‚Äòlabels‚Äô or ‚Äòvalue‚Äô key are present in the incoming ‚Äòdata‚Äô dictionary, then the keys of the data dictionary are set as the labels and all the values stored by the keys will make up the values of the table.
